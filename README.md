# 🚗 Musab Dealership – Luxury Car Website

A modern, responsive luxury car dealership website built with HTML, CSS, and JavaScript.  
Designed with a premium black & gold aesthetic to reflect high-end automotive brands.

---

## ✨ Overview

This project showcases a fictional luxury car dealership featuring a smooth, modern UI and interactive car browsing experience. The focus is on clean design, performance, and a premium user experience.

Users can:
- Browse featured luxury vehicles
- View detailed car specifications
- Navigate seamlessly between pages without full reload experience feel
- Explore a dynamic and interactive design system

---

## 🖥️ Tech Stack

- HTML5
- CSS3 (Custom styling + animations)
- Vanilla JavaScript (no frameworks)
- Responsive Design principles

---

## 🎨 Design Highlights

- Luxury-inspired black & gold theme
- Smooth scroll animations
- Interactive car detail pages
- Clean typography using modern fonts
- Responsive layout for all devices
- Card-based inventory system

---

## 🚘 Features

- Dynamic car detail rendering using JavaScript
- URL-based navigation system (`?id=car`)
- Smooth scroll-to-section navigation
- Animated UI elements on scroll
- Image-based car showcase system
- Lightweight and fast performance

---

## 📂 Project Structure
/index.html → Homepage
/car.html → Car details page
/styles/ → CSS files
/JS/ → JavaScript logic
/Material/ → Images and assets


---


---

## ⚙️ How It Works

- Each car is stored in a JavaScript object database
- Car details are loaded dynamically based on URL parameters
- Navigation updates without full page reload behavior
- UI elements animate on scroll for a modern feel

---

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop screens
- Tablets
- Mobile devices
